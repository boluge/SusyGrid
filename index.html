SusyGrid RWD

screen size

SusyGrid est un grille RWD en sass. Cette grille est mobile first. Elle est par defaut découpé en 12 colonnes, il est possible de choisir sa largeur, le nombre de colonnes, mais aussi la largeur de la goutière. Pour savoir comment porcéder, regardez le chapitre Configuration

L'utilisation

<!-- Classe Container -->
<div class="container">

	<!-- Class col-md-xx -->
	<div class="col-md-6">... </div>
	<div class="col-md-6">...</div>
	
	<!-- Class clearfix -->
	<div class="clearfix"></div>
	
	<!-- Class pre & post -->
	<div class="pre-md-2 col-md-8 post-md-2">...</div>
	
</div>
<!-- Taille d'une colonne sur smartphone -->
<div class="col-6">...</div>

<!-- Taille d'une colonne sur phablette -->
<div class="col-xs-6">...</div>

<!-- Taille d'une colonne sur tablette -->
<div class="grid-sm-6">...</div>

<!-- Taille d'une colonne sur laptop -->
<div class="grid-md-6">...</div>

<!-- Taille d'une colonne sur desktop -->
<div class="grid-lg-6">...</div>

Configuration

Les Variables

Vous pouvez personnaliser la grille en définissant les valeurs que vous souhaitez auxles variables ci contre, avant d'importer _susy.sass

$screen-xs: 480px
$screen-sm: 768px
$screen-md: 992px
$screen-lg: 1200px
		
$columns:  12

$susy: (
  columns: 12,
  gutters: 1/4,
  math: fluid,
  output: float,
  gutter-position: inside,
  global-box-sizing: border-box,
  use-custom: (rem: true)
)
